---

## 📂 Installation & Local Setup

1.  **Clone the project:**
    ```bash
    git clone [https://github.com/DiegoRiveraTM/portfolio-astro.git](https://github.com/DiegoRiveraTM/portfolio-astro.git)
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run Dev Server:**
    ```bash
    npm run dev
    ```
4.  **Mobile Preview:**
    ```bash
    npx astro dev --host
    ```
